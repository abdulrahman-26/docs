## Message Leaderboard Overview

The message leaderboard system tracks member activity based on messages sent in your server. It provides visual leaderboards and allows administrators to manually adjust message counts for accurate tracking and moderation purposes.

### Key Features

<Steps>
  <Step title="Activity Tracking">
    Automatically tracks messages sent by members across all server channels
  </Step>
  <Step title="Visual Leaderboards">
    Display attractive leaderboard frames showing all stats
  </Step>
  <Step title="Manual Management">
    Add, remove, or set specific message counts for individual users
  </Step>
  <Step title="Timeframe Options">
    Track activity across different time periods (daily, weekly, monthly, all-time)
  </Step>
</Steps>

## Message Count Management

### Adding Messages

Increase a user's message count by a specified amount within a timeframe.

<CodeGroup>

```Unspecified Example
/messages add (user) (number) (timeframe)
```


```Unspecified Syntax
/messages add @freycikkk 100 weekly
/messages add 123456789012345678 50 daily
```


```Aliases
/m add @freycikkk 100 weekly
/m add 123456789012345678 50 daily
```

</CodeGroup>

### Removing Messages

Decrease a user's message count by a specified amount within a timeframe.

<CodeGroup>

```Unspecified Example
/messages remove (user) (number) (timeframe)
```


```Unspecified Syntax
/messages remove @freycikkk 25 monthly
/messages remove 123456789012345678 10 daily
```


```Aliases
/m remove @freycikkk 25 monthly
/m remove 123456789012345678 10 daily
```

</CodeGroup>

### Setting Message Count

Set a user's exact message count for a specific timeframe.

<CodeGroup>

```Unspecified Example
/messages set (user) (number) (timeframe)
```


```Unspecified Syntax
/messages set @freycikkk 500 alltime
/messages set 123456789012345678 0 weekly
```


```Aliases
/m set @freycikkk 500 alltime
/m set 123456789012345678 0 weekly
```

</CodeGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Leaderboard not updating - Message counts not changing">
    **Issue**: The message leaderboard displays outdated information or doesn't reflect recent activity.

    **Solutions**:

    - Check if the bot has permission to read messages in all channels
    - Verify the bot is online and functioning properly
    - Ensure messages aren't being sent in ignored channels
    - Try viewing a different timeframe to see if tracking is working
    - Test with a few messages and check if counts update
  </Accordion>
  <Accordion title="Cannot modify message counts - Permission errors">
    **Issue**: Getting permission denied when trying to add, remove, or set message counts.

    **Solutions**:

    - Confirm you have administrator permissions or extra owner/admin status
    - Verify the target user exists and is in the server
    - Check if you're using the correct user mention or ID format
    - Ensure the timeframe parameter is spelled correctly
    - Try with a smaller number to test if there are value limits
  </Accordion>
  <Accordion title="Leaderboard frame not displaying - Visual issues">
    **Issue**: The leaderboard command returns text instead of the styled frame image.

    **Solutions**:

    - Ensure the bot has "Attach Files" permission in the channel
    - Check if the bot has "Embed Links" permission for proper display
    - Verify your internet connection for loading the leaderboard frame
    - Try using the command in a different channel
    - Check if there are any channel-specific restrictions on bot images
  </Accordion>
</AccordionGroup>

<Warning>
  Manual message count adjustments affect leaderboard rankings and statistics. Use these commands responsibly and only when necessary for moderation or data correction purposes.
</Warning>
